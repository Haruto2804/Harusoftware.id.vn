Bảo mật Client-side là gì?

Hầu hết các dịch vụ mã hóa file trên internet yêu cầu bạn tải file lên máy chủ của họ. Điều này mang lại rủi ro rò rỉ dữ liệu cực kỳ lớn nếu máy chủ bị hack.

Với Aegis Crypt, toàn bộ logic mã hóa được tính toán bằng Web Crypto API ngay trên CPU điện thoại/máy tính của chính bạn. Máy chủ Hosting của chúng tôi chỉ đóng vai trò phân phối giao diện HTML tĩnh, hoàn toàn không có khả năng đọc hay can thiệp vào tệp tin của bạn.

Tiêu chuẩn AES-256-GCM

AES-GCM (Galois/Counter Mode) không chỉ mã hóa dữ liệu của bạn, nó còn gắn thêm một "nhãn xác thực" (Authentication Tag). Nhãn này giúp phát hiện lập tức bất kỳ sự chỉnh sửa nhỏ nào đối với tệp tin đã mã hóa, ngăn chặn triệt để tấn công can thiệp dữ liệu trên đường truyền.

              1. Mã hóa file client-side trên Aegis Crypt hoạt động như thế nào?
            

            

              Aegis Crypt sử dụng Web Crypto API được tích hợp
              sẵn trong trình duyệt của bạn để mã hóa tệp tin cục bộ bằng thuật
              toán AES-GCM 256-bit cao cấp nhất. Dữ liệu tệp
              tin và mật khẩu của bạn không bao giờ được gửi lên bất kỳ máy chủ
              nào, đảm bảo tính bảo mật tuyệt đối 100%.
            

          


          
          

            

              2. Làm thế nào để giải mã một file đã được mã hóa?
            

            

              Để giải mã, bạn chỉ cần chuyển sang tab Giải Mã,
              kéo thả tệp tin có đuôi .enc đã tải về trước đó vào
              hệ thống, nhập chính xác mật khẩu ban đầu và nhấn nút
              Giải Mã và Tải Về để hồi phục tệp tin nguyên bản.
            

          


          
          

            

              3. Làm thế nào tôi có thể hosting trang này lên Internet?
            

            

              Do ứng dụng chạy hoàn toàn ở Client-side và là cấu trúc trang tĩnh
              (Single-page static HTML), bạn có thể dễ dàng hosting nó miễn phí
              100% bằng các dịch vụ cực kỳ mạnh mẽ như
              GitHub Pages, Vercel, Netlify, hoặc Cloudflare Pages. Chỉ cần tải duy nhất file index.html này lên là
              trang web sẽ hoạt động mượt mà.
